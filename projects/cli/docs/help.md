`fs help`
=========

Display help for fs.

* [`fs help [COMMAND]`](#fs-help-command)

## `fs help [COMMAND]`

Display help for fs.

```
USAGE
  $ fs help [COMMAND...] [-n]

ARGUMENTS
  [COMMAND...]  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for fs.
```
